# ADR 模板

> ADR = Architecture Decision Record（架构决策记录）

## 使用方法

每次做出重要技术决策时，复制此模板创建新文件：

```
docs/decisions/NNN-<short-title>.md
```

例如：`002-use-postgresql-over-sqlite.md`
